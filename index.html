Strings

  • A double quoted string can contain single quote and vice versa
  • To span a text in multiple lines either put it in triple quotes or use \ to break it
  • Python strings are immutable
  • The '+' operator can concatenate two strings
  • Characters in a string can be accessed using the standard [ ] syntax
  • The len(string) function returns the length of a string
  • The str(integer) function converts values to a string form so they can be combined with other strings
  • s.lower(), s.upper() -- returns the lowercase or uppercase version of the string
  • s.strip() -- returns a string with whitespace removed from the start and end
  • s.isalpha() -- Return True if all characters in S are alphabetic and there is at least one character in s, False otherwise.
    similar methods: s.isdigit() and s.isspace()
  • s.split('delim') -- returns a list of substrings separated by the given delimiter
  • s.join(list) -- opposite of split(), joins the elements in the given list together using the string as the delimiter
  • s.replace('old', 'new') -- returns a string where all occurrences of 'old' have been replaced by 'new'
  • s.startswith('other'), s.endswith('other') -- tests if the string starts or ends with the given other string
  • s.find('other') -- return index of occurence of other in s
  • s.find('substring',startPosition,endPosition)
  • s.slice[start:end] is used to slice a string
  • negative numbers can be used to index string from the end
  • formatted strings -- "%s has %f rating on IMDB" % ("Dead Poets Society",8.1)
  • add a character to string at a position through slicing -- string[:position] + character + string[position+1:]
  • list of lowercase characters
    import string
    alphabets = string.ascii.lowercase
  • s.title() will capitalize first character of each word in the string
    s.capitalize() will just capitalize the first character in the string
  • remove duplicates from a string 
    					   def unique_list(l):
    						ulist = []
    						[ulist.append(x) for x in l if x not in ulist]
    						return ulist

List

  • list.append(elem) -- adds a single element to the end of the list. Common error: does not return the new list, just modifies the original.
  • list.insert(index, elem) -- inserts the element at the given index, shifting elements to the right.
  • list.extend(list2) adds the elements in list2 to the end of the list. Using + or += on a list is similar to using extend().
  • list.index(elem) -- searches for the given element from the start of the list and returns its index. Throws a ValueError if the element does not appear (use "in" to check without a ValueError).
  • list.remove(elem) -- searches for the first instance of the given element and removes it (throws ValueError if not present)
  • list.sort() -- sorts the list in place (does not return it). (The sorted() function shown later is preferred.)
  • list.reverse() -- reverses the list in place (does not return it)
  • list.pop(index) -- removes and returns the element at the given index. Returns the rightmost element if index is omitted (roughly the opposite of append())
  • convert a string to list -- l = list(string)

Sorting

  • sorted(iterable) -- returns a sorted list, to sort in reverse use -- sorted(iterable,reverse=True)
  • sorted(list,key=len) -- will sort list according to the length of individual strings
  • sorted(list,key=str.lower) -- will treat uppercase and lowercase as the same

Sets

  • A set is an unordered collection of elements without duplicate entries. When printed, iterated or converted into a sequence, its elements will appear in an arbitrary order. Example: if you want to find average of all the distinct numbers in a list then use:
    sum(set(array))/len(set(array))
  • Creating sets 
    	>> myset = {1, 2} # Directly assigning values to a set
	>> myset = set()  # Initializing a set
	>> myset = set(['a', 'b']) # Creating a set from a list
    Modifying sets
    	>> myset.add('c')
	>> myset
	{'a', 'c', 'b'}
	>> myset.add('a') # As 'a' already exists in the set, nothing happens
	>> myset.add((5, 4))
	>> myset
	{'a', 'c', 'b', (5, 4)}
    Modifying sets using update function
    	>> myset.update([1, 2, 3, 4]) # update() only works for iterable objects
	>> myset
	{'a', 1, 'c', 'b', 4, 2, (5, 4), 3}
	>> myset.update({1, 7, 8})
	>> myset
	{'a', 1, 'c', 'b', 4, 7, 8, 2, (5, 4), 3}
	>> myset.update({1, 6}, [5, 13])
	>> myset
	{'a', 1, 'c', 'b', 4, 5, 6, 7, 8, 2, (5, 4), 13, 3}
    • Removing items Both the discard() and remove() functions take a single value as an argument and removes that value from the set. If that value is not present, discard() does nothing, but remove() will raise a KeyError exception. 
    	>> myset.discard(10)
	>> myset
	{'a', 1, 'c', 'b', 4, 5, 7, 8, 2, 12, (5, 4), 13, 11, 3}
	>> myset.remove(13)
	>> myset
	{'a', 1, 'c', 'b', 4, 5, 7, 8, 2, 12, (5, 4), 11, 3}
    Common Set Operations Using union(), intersection() and difference() functions. 
    	>> a = {2, 4, 5, 9}
	>> b = {2, 4, 11, 12}
	>> a.union(b) # Values which exist in a or b
	{2, 4, 5, 9, 11, 12}
	>> a.intersection(b) # Values which exist in a and b
	{2, 4}
	>> a.difference(b) # Values which exist in a but not in b
	{9, 5}
    The union() and intersection() functions are symmetric methods: 
    	>> a.union(b) == b.union(a)
	True
	>> a.intersection(b) == b.intersection(a)
	True
	>> a.difference(b) == b.difference(a)
	False

Formatting

  • Note: number after colon ":" represents total characters
  • simple formatting -- '{}    {}'.format('Atul', 'Bhatt') -- the output will have 3 spaces between "Atul" and "Bhatt".
  • right align your value -- '{:>10}'.format('test') -- test will have 6 spaces before it.
  • left align your value -- '{:10}'.format('text') -- test will have 6 spaces after it.
  • you can also specify the character instead of spaces -- '{:_<10}'.format('text') -- text will have 6 underscores after it
  • center align your string -- '{:^10}'.format('text')
  • truncate a string -- '{:.5}'format('thinkingfast') -- it will only print think
  • It is possible to combine truncating and padding '{:_>10.5}'.format('thinkingfast'} -- it will print five underscores and then think
  • padding with numbers -- '{:10d}'.format(15) -- 15 will have 8 spaces before it
  • padding with numbers -- '{:>010d}'.format(15) -- it will print 8 zeros before 15
  • padding with float -- '{:06.2f}'.format(15.34342232) -- two digits after decimal point, total 6 characters with zeroes from left
  • positive and negative with integers -- '{:+d}'.format(12) -- it will print +12
  • space between character and value -- '{:=+10d}'.format(12) -- it will print 7 spaces between + and 12
  • Date and time formatting
    from datetime import datetime
    '{:%Y-%m-%d %H:%M}'.format(datetime(2001, 2, 3, 4, 5))
    Output 2001-02-03 04:05
  • print binary equivalent of a number -- '{0:b}'.format(23)
    similarly use o for octal, x (lowercase) for lowercase hexadecimal X (uppercase) for uppercase hexadecimal

Slicing

  • a[start:end] -- # items start through end-1
    a[start:] ------ # items start through the rest of the array
    a[:end] ------- # items from the beginning through end-1
    a[:] ----------- # a copy of the whole array
  • There is also the step value, which can be used with any of the above:
    a[start:end:step] # start through not past end, by step
  • a[-1]     # last item in the array
    a[-2:]     # last two items in the array
    a[:-2]     # everything except the last two items
  • Similarly, step may be a negative number:
    a[::-1]    # all items in the array, reversed
    a[1::-1]    # the first two items, reversed
    a[:-3:-1]     # the last two items, reversed
    a[-3::-1]    # everything except the last two items, reversed
